## Overview

This project is a simple calculator implemented in C for the **Tiva C Series LaunchPad** as part of the ELEC3662 course at the University of Leeds in 2023. The calculator performs basic arithmetic operations including addition, subtraction, multiplication, and division using the onboard hardware.

The aim of the project was to gain hands-on experience with embedded systems programming, interfacing with microcontroller peripherals, and implementing user input/output logic on the Tiva LaunchPad.

## Features

* Addition, subtraction, multiplication, and division
* User input through the onboard switches
* Display output via the onboard LEDs or serial terminal (depending on implementation)
* Simple and efficient C code suitable for embedded systems

## Hardware

* **Tiva C Series LaunchPad (TM4C123GH6PM)**
